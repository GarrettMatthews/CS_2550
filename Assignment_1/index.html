2048

2048 is a game of doubling. When presented with a grid, typically 4x4, numbers will randomly appear, either a 2 or a 4. Moving these numbers left, right, up or down, you can combine like numbers to create a larger number. For example 2 and 2 would be 4, 16 and 16 would be 32, 128 and 128 would be 256 and so on and so forth. The goal of the game is to reach 2048 without running out of possible moves. Can you do it?
